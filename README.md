## 今後の更新手順

### プロフィール写真を追加する場合

1. 画像ファイルを用意（推奨: 400×400px以上の正方形、JPG or PNG）
2. HPフォルダに画像を配置
3. index.html を編集:

```html
<!-- 変更前 -->
<div class="profile-placeholder">
  <span>Photo</span>
</div>

<!-- 変更後 -->
<img src="profile.jpg" alt="Hayami" class="profile-photo">
```

4. styles.css に追加:

```css
.profile-photo {
  width: 200px;
  height: 200px;
  border-radius: 50%;
  object-fit: cover;
  border: 4px solid var(--color-accent-light);
}

@media (max-width: 768px) {
  .profile-photo {
    width: 150px;
    height: 150px;
  }
}

@media (max-width: 480px) {
  .profile-photo {
    width: 120px;
    height: 120px;
  }
}
```

5. GitHubにプッシュ:

```bash
cd "C:\Users\kumamon\OneDrive\Desktop\HP"
git add .
git commit -m "Add: プロフィール写真を追加"
git push
```

### 実績を追加する場合

実績が用意できたら、「できること」セクションにアコーディオン機能を追加予定。

#### 必要な情報
1. カテゴリ（業務自動化 / AI活用 / 教育子育て / LINE・Google連携）
2. タイトル（例：「出欠確認の自動集計システム」）
3. YouTubeのURL（埋め込み用）
4. 簡単な説明（1〜2文、任意）

#### 効果的なポートフォリオの示し方（推奨）

| 要素 | 役割 |
|------|------|
| ビフォーアフター | 効果を一目で伝える |
| 数字 | 説得力を高める（例：月10時間→0時間） |
| 利用者の声 | 信頼感を与える |
| スクリーンショット or 動画 | 実物を見せる |

#### 実績の構成例

```
┌────────────────────────────────────┐
│ 📊 出欠確認の自動集計システム        │

│ Before: 毎朝30分かけて手集計        │
│ After:  自動集計で作業時間ゼロ      │
│                                    │
│ [スクリーンショット or YouTube動画]  │
│                                    │

│    余裕ができました」― 小学校教員   │
```
